# GLYPH TOOL

A lightweight utility for working with Unicode characters (emojis & text symbols) in Minecraft: Bedrock Edition.  
More information: [Bedrock Wiki — Emojis & Text Symbols](https://wiki.bedrock.dev/concepts/emojis.html)

## Fork
This repository is a personal fork of the original Glyph Tool by NhanAZ, adapted to personal preferences and workflows. See the commit history for details on changes.
